Data visualizations on the Web can be klugey--too many browser anomalies and versions to deal with. D3.js to the rescue! Itâ€™s a Javascript library built so that developers can create embedded charts and graphs using web standards--HTML, SVG, and CSS--without fussing over proprietary nonsense. The D3 Project (Data-Driven Documents) gives you reusable graph components and complete charts that are fully customizable--no boilerplate code here!

Charts included are:

  • Simple Line
  • Scatter / Bubble
  • Stacked / Stream / Expanded Area
  • Discrete Bar
  • Grouped / Stacked Multi-Bar
  • Horizontal Grouped Bar
  • Line and Bar Combo
  • Cumulative Line
  • Line with View Finder
  • Pie Chart
  • Bullet Chart
  • HTML Indented Tree
